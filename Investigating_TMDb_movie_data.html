Introduction¶

The TMDb dataset contains information about 10,000 movies collected from the The Movies Database. This includes information such as the movie budget, the revenue it generated, the title of the movie, its cast, its genre and release year along with various other features. The aim is to analyze this dataset to answer a bunch of questions such as;

  1. Which movies have generated a maximum and minimum revenue?
    • 1.1 Which movie has generated a maximum profit?
    • 1.2 Which movie has generated a minimum profit?
  2. Which genre has highest release of movies?
  3. Which genre has the highest popularity?
  4. Is the release year related to profits?
  5. What is the average runtime of movies in each year?
  6. What is the runtime of the movies having generated profit greater than the average profit?

Data Cleaning¶

Observations¶

  1. Columns like id, imdb_id, homepage, tagline, overview, budget_adj, revenue_adj, vote_average, vote_count, keywords, director, cast, production_companies can be deleted as they will not be used for analysis.
  2. The release_date column is of a string datatype. We need to convert it into a date time format.
  3. We also need to remove rows having null values in any of the columns as all columns are important for our analysis and they cannot be left blank.
  4. In the dataset, the revenue and budget columns have many values as 0. We can replace these values by the mean of the column.
  5. As seen in the dataset, we assume the currency used for columns such as budget and revenue is US dollars.

From the above visualizations, we derive a few insights into the columns data. Here is a brief description of each column we will be using for our analysis;

  1. popularity: This column represents how popular the movie was. The range of the data is majorly between 0 and 10 as seen in the plot above, 0 being the least popular movie and 10 being the most popular movie.
  2. budget: This column represents the overall budget of the movie. We assume that the currency is in USD(\$). There is a bit of error in this column which is rectified in the section below.
  3. revenue: This column represents the overall revenue earned of the movie. We assume that the currency is in __USD(\$). There is a bit of error in this column which is rectified in the section below.
  4. runtime: The runtime of the movie is its duration in minutes. We can observe that the duration is mostly between 0 to close to 190.
  5. release_year: This column represents the year in which the movie was released.
  6. original_title: This column contains the title of the movie.
  7. release_date: This column represents the exact release date of the movie.
  8. genre: This column represents the movie genre. The number of total genres are found out in the analysis below.

    #concatenate all the rows of the genres.

    #conts each of the genre and return.
    print(data.unique())
    info = data.value_counts(ascending=False)

Conclusions¶

  1. Which movies have generated a maximum and minimum revenue? The movie Avatar has generated a maximum revenue of \$2781505847 USD.
    The movie Shattered Glass has generated a minimum revenue of \$2 USD.

    • 1.1 Which movie has generated a maximum profit?

      The movie Avatar has generated a maximum profit of \$2544505847 USD.

    • 1.2 Which movie has generated a minimum profit?

      The movie The Warrior's Bay has generated a minimum profit of -\$413912431 USD.

    2. In order to calculate a movie with minimum revenue, we first found out the minimum value of revenue in the revenue column by using the min method and stored it in a variable min_revenue. We then found out the row corresponding to the min_revenue and obtained the results.

    4. In order to calculate a movie with minimum profit, we used the min method to find out the minimum profit and stored it in a variable min_profit. We then found out the row corresponding to the min_profit and obtained the results.



  1. Which genre has highest release of movies? The genre Drama has the highest release of movies - 4365 movies.

Limitations of the analysis¶

  1. There were many rows in the dataset where the revenue and budget had values - 0. These values were imputed with the means of their respective columns. In our analysis, budget and revenue columns had zero values which were then replaced with the mean of the budget column and the revenue column respectively.
  2. An assumption was made on the revenue and budget columns that the currency used for them is USD(\$). It can also be possible that the budget for a movie in a different country might have a different currency. This causes disparity in our analysis and might result in our entire analysis going wrong.
